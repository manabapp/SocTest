SocTest is the mobile app that communicates via socket with using system calls to the kernel (Darwin based on Unix). You can access to various hosts on the Internet with 3G/4G, and access to servers on secured organization's intranet with Wi-Fi.



All of network operations in SocTest is implemented with POSIX C API. It is designed to delegate processing to the kernel as much as possible without sandwiching any highly abstract API, without error judgment of user input more than necessary.
As a result, it is realizable a basic (low-level) and interactive (high-flexibile) socket communication. You can complement the major network test tools in troubleshooting and connectivity testing, etc.



Through using SocTest, you will become able to imagine the socket behavior, the basis of network technology. I hope that you utilize not only as a test tool but also as a learning tool to deepen your understanding of network.
